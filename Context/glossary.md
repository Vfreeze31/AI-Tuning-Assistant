# Knowledge Node
A single concept.

# Chunk
A semantically complete section of a document.

# Canonical Document
The Git markdown source.

# Generated Data
Anything created from canonical documents.

# Metadata
Machine-readable information describing a document.

# Vehicle Platform
Manufacturer chassis/platform.

# Vehicle Family
Cars sharing common engineering.

# Tuning Principle
A general engineering concept independent of any game.

# Game Profile
A game's implementation of tuning parameters.
